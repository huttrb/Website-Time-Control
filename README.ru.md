# Website Time Control

> Компактное расширение для Chrome, которое считает время на сайтах, показывает динамику и помогает держать правила отслеживания под контролем.

## Обзор

Website Time Control отслеживает, сколько времени вы проводите в активных вкладках браузера, и хранит статистику локально. В popup есть быстрый список сайтов, фильтры по датам, избранное, настройки правил отслеживания и отдельная страница статистики со сводками и графиками.

## Главное

| Раздел | Что делает |
| --- | --- |
| Отслеживание | Считает время активной вкладки и сохраняет его по сайту или пользовательскому правилу. |
| Фильтры | Показывает сегодня, вчера, неделю, месяц, всё время и пользовательский диапазон дат. |
| Статистика | Отдельная страница с общим временем, средним за день, топом сайтов, доменами, динамикой по дням, лучшим днём, серией и последней активностью. |
| Правила | Поддерживает правила с названием, доменом, URL-паттерном и состоянием включения. |
| Избранное | Закрепляет важные сайты выше в основном списке popup. |
| Приватность | Хранит настройки и статистику в `chrome.storage.local`. |

## Экраны

### Popup

- Текущая активная вкладка с favicon и временем.
- Сортировка и фильтры периода.
- Переключатель избранного и быстрые действия удаления.
- Календарь с подсветкой дней, где есть статистика.

### Страница статистики

Открывается по иконке графика в правом верхнем углу, рядом с настройками.

- Общее время и среднее за активный день.
- Время на избранных сайтах.
- Самая длинная серия активных дней.
- Горизонтальный выбор периода: сегодня, вчера, неделя, месяц, всё время и пользовательский диапазон дат.
- График активности с компактными числами дней, подсказкой при наведении и календарём, где подсвечены дни со статистикой. Для длинных периодов график фокусируется на последних 14 днях.
- Топ сайтов с процентными полосами.
- Разбивка по доменам.
- Лучший день, сайт-лидер, количество сайтов и последняя активность.

### Настройки

- Выбор языка.
- Отслеживание всех сайтов или только выбранных правил.
- Минимальное время для показа элемента.
- Добавление, редактирование, включение, отключение и удаление отслеживаемых сайтов.
- Очистка всей сохранённой статистики.

## Как работает отслеживание

Расширение слушает смену активной вкладки и сохраняет прошедшее время для текущего отслеживаемого сайта.

Если **Отслеживать только выбранные сайты** выключено:

- Могут отслеживаться любые обычные сайты.
- Если URL совпал с правилом из списка, время сохраняется под стабильным ключом этого правила и отображается с его названием.
- Если совпадения нет, время сохраняется по домену страницы.

Если **Отслеживать только выбранные сайты** включено:

- Отслеживаются только включённые элементы из списка.
- Вкладки, которые не совпали ни с одним включённым правилом, игнорируются.

Правила сопоставляются по URL-паттерну и домену. Название используется только как подпись, поэтому два разных домена могут иметь одинаковое отображаемое имя без объединения статистики.

## Правила отслеживания

Каждая запись содержит:

| Поле | Назначение |
| --- | --- |
| `Name` | Название, которое показывается в popup и на странице статистики. |
| `Domain` | Домен для favicon и fallback-сопоставления. |
| `Pattern` | Фрагмент URL для более точного совпадения. |
| `Enabled` | Включено ли правило. |

Более специфичные URL-паттерны проверяются раньше широких совпадений по домену.

## Язык

Расширение поддерживает русский и английский язык.

При первом запуске язык определяется автоматически по языку интерфейса браузера. Если язык начинается с `ru`, используется русский; во всех остальных случаях используется английский. После ручного выбора языка в настройках он сохраняется и используется при следующих запусках.

## Установка в Chrome

1. Выполните `npm run build`.
2. Откройте `chrome://extensions`.
3. Включите **Developer mode**.
4. Нажмите **Load unpacked**.
5. Выберите папку `dist/`.

## Permissions

Расширение использует:

- `tabs`: чтобы получать активную вкладку, URL, заголовок и favicon.
- `storage`: чтобы локально хранить настройки и статистику.

## Приватность

Статистика использования сайтов хранится локально в браузере через `chrome.storage.local`. В проекте нет внешнего backend-сервиса или синхронизации данных.
