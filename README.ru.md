# Website Time Control

Расширение для Chrome, которое отслеживает, сколько времени вы проводите на сайтах. Оно показывает статистику по сайтам, поддерживает избранное, фильтры по датам, пользовательские правила отслеживания и компактный popup-интерфейс.

[English version](README.md)

## Возможности

- Отслеживание времени активной вкладки.
- Фильтры: сегодня, вчера, неделя, месяц, всё время и пользовательский диапазон дат.
- Подсветка доступных дат в календаре, если за эти дни есть статистика.
- Добавление сайтов в избранное.
- Отслеживание всех сайтов или только выбранных.
- Пользовательские правила с названием, доменом, паттерном и флагом включения.
- Разные домены могут иметь одинаковое отображаемое имя без конфликта статистики.
- Локальное хранение настроек и статистики через `chrome.storage.local`.
- Manifest V3.

## Как работает отслеживание

Расширение слушает смену активной вкладки и сохраняет прошедшее время для текущего отслеживаемого сайта.

Если **Отслеживать только выбранные сайты** выключено:

- Могут отслеживаться любые обычные сайты.
- Если URL совпал с правилом из списка, время сохраняется под стабильным ключом этого правила и отображается с его названием.
- Если совпадения нет, время сохраняется по домену страницы.

Если **Отслеживать только выбранные сайты** включено:

- Отслеживаются только включённые элементы из списка.
- Вкладки, которые не совпали ни с одним включённым правилом, игнорируются.

Название правила используется только как подпись. Ключ статистики строится отдельно, поэтому два разных домена могут иметь одинаковое имя.

## Правила отслеживания

Каждая запись содержит:

- **Name**: название, которое показывается в popup.
- **Domain**: домен для favicon и fallback-сопоставления.
- **Pattern**: часть URL для более точного совпадения.
- **Enabled**: включено ли правило.

Более специфичные URL-паттерны проверяются раньше широких совпадений по домену.

## Язык

Расширение поддерживает русский и английский язык.

При первом запуске язык определяется автоматически по языку интерфейса браузера. Если язык начинается с `ru`, используется русский; во всех остальных случаях - английский. После ручного выбора языка в настройках он сохраняется и используется при следующих запусках.

## Установка в Chrome

1. Выполните `npm run build`.
2. Откройте `chrome://extensions`.
3. Включите **Developer mode**.
4. Нажмите **Load unpacked**.
5. Выберите папку `dist/`.

## Технологии

- TypeScript
- Pinia
- Vue Router
- Tailwind CSS
- Vite
- Chrome Extensions Manifest V3

## Permissions

Расширение использует:

- `tabs`: чтобы получать активную вкладку, URL, заголовок и favicon.
- `storage`: чтобы локально хранить настройки и статистику.

## Privacy

Статистика использования сайтов хранится локально в браузере через `chrome.storage.local`. В проекте нет внешнего backend-сервиса или синхронизации данных.
