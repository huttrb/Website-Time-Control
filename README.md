Как работает "Отслеживать только выбранные сайты":

***Если выключено:*** расширение трекает любые сайты. Если URL совпал с правилом из списка, пишет время под его name, например video; если не совпал, пишет по домену.
***Если включено:*** расширение трекает только те сайты/паттерны, которые есть в списке и включены чекбоксом. Все остальные вкладки игнорируются и время по ним не пишется.
